# rtb-chart

## Description

Fontend part of the project rtb-chart. This project features data fetching from a persistence layer and renders a table with the fetched data. It also features filters and pagination, along with responsive design.

## Installation

### Zip File

#### Requirements before installation

##### Tested and developed using Node version 18.12.0, but also works with Node version 16

```bash
# Install dependencies
npm install

# or
yarn
```

to run project in development mode

```bash
npm run dev

# or
yarn dev

# will be running on port: 3000 - access: <http://localhost:3000>
```

### Git Repository

```bash
# clone this repository with ssh:
git clone <git@github.com:Isadora-Costa/rtb-chart.git>

# or clone this repository with url:
git clone <https://github.com/Isadora-Costa/rtb-chart.git>

# Access the project folder in the terminal path/folder
cd rtb-chart

# Install dependencies
npm install

# or
yarn
```

to run project in development mode

```bash
npm run dev

# or
yarn dev

# will be running on port: 3000 - access: <http://localhost:3000>
```
